Salutes from Araraquara!

Well, I was looking for a way to port my old blog from WordPress to GitHub Pages. After a little research I came around [`Jekyll`](https://jekyllrb.com/), and I'm just in love with this thing...

After following a simple quickstart guide and creating a new GitHub repo, here you go. My first post on my brand new blog! :) 

The best part is I can finally use markdown to write my blog posts, but anyway... This is intended to be a blog on all things related to Android security engineering, but I also intend to post about electronics, Swiss Army Knife hacks, woodworking, or whatever programming or DIY pet project I might be working on that I find worth sharing. 

If anyone ever reads this, I truly hope you enjoy! _Tchau!_
